* Git Commands
- git rm <file> : delete file using git bash.
- git restore --staged <file>
- git restore <file> : To restore the deleted files.
- git diff : To check the what chnges has been made.
- git rebase --skip/abort : To skip or abort the reabasing.
- git merge --abort : To abort the merging.
- rm -rf .git : This command will remove the reference to git.
- git show : show one or more tages and commits.
- git show --name-only : show the list of files modified after the commit information.

* These commands will bring up the below interface with options to perform the following:
- p, pick = use commit.
- r, reword = use commit, but edit the commit message.
- e, edit = use commit, but stop for amending.
- s, squash = use commit, but meld into previous commit.
- f, fixup = like “squash”, but discard this commit’s log message.
- x, exec = run command (the rest of the line) using shell.
- d, drop = remove commit.

- git push origin --delete <Branch> : To delete remote branch use this command.
